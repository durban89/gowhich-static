Nest.js如何安装

这个过程其实官网也有 - [点击这里](https://docs.nestjs.com/first-steps)

这里做个简单的概述

1、安装nodejs

2、全局安装nest.js，命令如下

```bash
npm i -g @nestjs/cli
```

3、创建项目

```bash
nest new project-name
```

是不是很简单

4、运行项目

```bash
npm run start
```

如果是开发的话

```bash
npm run start:dev
```
